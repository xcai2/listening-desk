01.A

The 45-Minute Daily Routine

四个时间块,合计 45 分钟。每天完成一块就打勾 — 完成后会记入连续打卡天数。

Last 30 days · 打卡记录
02

The Library

预先整理的真实素材 — earnings calls、Bloomberg、FP&A 面试、商务会议、日常英语。每个素材都有来源链接和精挑的 chunks。

03

The Shadowing Drill

挑场景 → 出现一句 → 大声跟读 5 遍 → 点 Next。把这套循环每天 15 分钟坚持下来,口音和流畅度会显著变化。

04

Chunks Notebook

你专属的语块库。从 Library / Shadowing 加进来,或者点 + 自己加。能搜索,能按场景过滤。

05

Add a New Source

添加一个新的 YouTube 频道、podcast、或公司的 earnings call。填完后会进入 Library,可以从里面手动添加 chunks。

名字写得具体一点,方便以后找
YouTube / Podcast / 文章链接 — 想加多个的话,先建一个,之后还能在 Library 里编辑
06

The Reflection Desk

三个模式：每天复盘用英文讲述今天的事 → 场景练习按情景跟读并说出英文表达 → 历史查看过往记录。

今天发生了什么？

用英文描述今天的 3 件事，每件 30–60 秒。
🎙 录音 / 输入
支持 Chrome / Edge
🔍 发现的表达问题

今日提示 · Prompts

选一个方向练，或者自由讲。
Event 1 Describe something that happened at work or in class today. What was the situation? What did you say or do?
Event 2 Was there a moment today where you struggled to express something in English? What were you trying to say?
Event 3 What's one thing you want to remember to say differently next time? Write a better version.
选择一个场景 → 看中文 → 先自己说 / 写出英文 → 点击显示正确表达
